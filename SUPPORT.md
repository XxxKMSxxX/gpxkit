# Support

- **Bug reports** — [open an issue](https://github.com/XxxKMSxxX/gpxkit/issues).
  A description of the GPX file that reproduces the problem (point count,
  which tags it has) helps enormously.
- **Questions and ideas** —
  [GitHub Discussions](https://github.com/XxxKMSxxX/gpxkit/discussions).
- **Security vulnerabilities** — see [SECURITY.md](./SECURITY.md) for private
  disclosure. Please do not open public issues for these.
